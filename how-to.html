Add a new plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap the plus button at the right bottom corner. This opens the screen for adding a new plant.
  3. Write plant name in 'Plant name' field.
  4. Add plant photo using either the Gallery or Camera button.
  5. Save the plant using check mark button at the top right corner.

Edit name and photo of a plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant whose name or photo you want to change. This will open the plant actions list.
  3. Tap the overflow (triple dot) menu in top right corner and select 'Edit Plant'. This will open the 'Edit plant' screen.
  4. Write the new name, if you want to edit the name.
  5. Choose a new photo, if you want to edit the photo.
  6. Save the plant using the check mark button in top right corner.

Delete a plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant that you want to delete. This will open the plant actions list.
  3. Tap the overflow (triple dot) menu in top right corner and select 'Delete Plant'.
  4. Tap Delete button in confirmation dialog.

Add a plant action

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant for which you want to add an action. This will open the plant actions list screen with the name of that plant as the title.
  3. In plant action list screen, tap the plus button in the bottom right corner to open the dialog for plant action name.
  4. Enter the name of the plant action and tap Add button.
