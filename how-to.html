Add a new plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap the plus button at the right bottom corner. This opens the screen for adding a new plant.
  3. Write plant name in 'Plant name' field.
  4. Add plant photo using either the Gallery or Camera button.
  5. Save the plant using check mark button at the top right corner.

Edit name and photo of a plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant whose name or photo you want to change. This will open the plant actions list.
  3. Tap the overflow (triple dot) menu in top right corner and select 'Edit Plant'. This will open the 'Edit plant' screen.
  4. Write the new name, if you want to edit the name.
  5. Choose a new photo, if you want to edit the photo.
  6. Save the plant using the check mark button in top right corner.

Delete a plant

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant that you want to delete. This will open the plant actions list.
  3. Tap the overflow (triple dot) menu in top right corner and select 'Delete Plant'.
  4. Tap Delete button in confirmation dialog.

Add a plant action

  1. Go to the home screen that has the title 'Your plants'.
  2. Tap on the plant for which you want to add an action. This will open the plant actions list screen with the name of that plant as the title.
  3. In plant action list screen, tap the plus button in the bottom right corner to open the dialog for plant action name.
  4. Enter the name of the plant action and tap Add button.

Edit a plant action name

  1. Go to the action list screen of the plant.
  2. Tap the calendar button of that action to open the history screen.
  3. In the history screen toolbar, tap pencil button to open the edit dialog.
  4. Enter the name of the plant action and tap Update button.

Delete a plant action

  1. Go to the action list screen of the plant.
  2. Tap the calendar button of that action to open the history screen.
  3. In the history screen toolbar, tap bin button to open the confirmation dialog.
  4. In the confirmation dialog tap the delete button.

Record a plant action

  1. Go to the action list screen of the plant.
  2. Tap the big checkmark button of the action that you want to perform.
  3. This will mark that action as done and the app will automatically generate the date and time history for you.

Edit date and time of an action

  1. Go to the action list screen of the plant.
  2. Tap the pencil button of the action that you want to edit the date and time for. This will open the calendar to pick date.
  3. Select the date and tap OK to open the clock.
  4. Select the time and tap OK.

Add a note for a plant

  1. Go to the action list screen of the plant.
  2. Tap the note button in the toolbar to open the notes screen for that plant.
  3. Write your note.
  4. Tap checkmark button in the toolbar to save the note.

Set reminder for an action

  1. Go to the action list screen of the plant.
  2. Tap the alarm button of the action that you want to set the reminder for. This will open the alarm dialog.
  3. Pick the number of days after which to repeat the alarm.
  4. Tap Done.

Postpone a reminder

  1. Go to 'Your reminders' screen.
  2. Tap the snooze button of the action that you want to postpone the reminder for. This will open Postpone Alarm dialog.
  3. Pick the number of days to postpone the reminder.
  4. Tap Done.

Disable reminder for an action

  1. Go to the action list screen of the plant.
  2. Tap the alarm button of the action that you want to disable the reminder for. This will open the alarm dialog.
  3. Pick zero to disable the alarm.
  4. Tap Done.

Enable notification for reminders

  1. Swipe right from the left edge of the screen to open the navigation menu. You can also open the navigation menu by going to 'Your plants' screen and tapping the hamburger button.
  2. Tap the Settings to open the settings screen.
  3. Tap the toggle button of the Action Reminder setting to the enabled position.

Disable notification for reminders

  1. Swipe right from the left edge of the screen to open the navigation menu. You can also open the navigation menu by going to 'Your plants' screen and tapping the hamburger button.
  2. Tap the Settings to open the settings screen.
  3. Tap the toggle button of the Action Reminder setting to the disabled position.

Change notification time for reminders

  1. Swipe right from the left edge of the screen to open the navigation menu. You can also open the navigation menu by going to 'Your plants' screen and tapping the hamburger button.
  2. Tap the Settings to open the settings screen.
  3. Tap Action Reminder Time setting to open the clock.
  4. Select the new time for notification and tap OK.

Access action history

  1. Go to the action list screen of the plant.
  2. Tap the calender button of the action that you want to access the history for. This will open the action history screen.

Backup plant data

  1. Swipe right from the left edge of the screen to open the navigation menu. You can also open the navigation menu by going to 'Your plants' screen and tapping the hamburger button.
  2. Tap Backup to open the Backup screen.
  3. Tap the Backup button to open file explorer.
  4. Select the folder in which you want to store the backup file.
  5. Tap the Save button in the bottom right corner.

Restore plant data

  1. Swipe right from the left edge of the screen to open the navigation menu. You can also open the navigation menu by going to 'Your plants' screen and tapping the hamburger button.
  2. Tap Restore to open the Restore screen.
  3. Tap the Restore button to open file explorer.
  4. Navigate to the folder where you previously saved the backup file.
  5. Tap the backup file to start the restore operation. If you are selecting the file from the cloud like Google Drive, the progress will stay on zero while the file is being downloaded to your device.
